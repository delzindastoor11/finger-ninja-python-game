A Python-based Fruit Ninja style game using:
- Pygame for rendering
- OpenCV for camera input
- MediaPipe for hand tracking

## Features
- Real-time hand tracking
- Camera-based slicing mechanic
- Randomized fruit spawning
- Score tracking

## Requirements
- Python 3.11 (recommended)
- Webcam

## Installation

pip install pygame mediapipe opencv-python

Notes
MediaPipe currently does not fully support Python 3.13.
For best compatibility, use Python 3.11.
